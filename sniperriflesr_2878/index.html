푸딩의 날

* 2010년 8월, 오하요 유업(オハヨー乳業)에서 제정한 날. 빼빼로 데이랑 비슷한 개념. 

날짜는 「푸딩을 먹으면 나도 모르게 빙긋(プリンを食べると、思わずにっこり)」라는 문장에서 유래되었다.

매년 25일인 이유는 저 문장 뒤의 "빙긋(にっこり)"에서 2(に), 5(ご = こ)라고 읽히기 때문이다.



원제 : ひまくりカレンダー 2014/02/25　プリン


서클 : 破り処


작가 : ハカノシンシ [ 하카노 신시 ]


역자 : Helios

식자 : Luftwaffe


출연 : 카기야마 히나 [ 꼬마 ]
